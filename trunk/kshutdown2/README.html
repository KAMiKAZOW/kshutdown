KShutdown Help

KDE 4/Linux

Requirements

  1. KDE 4.0 and Qt 4.3
  2. CMake 2.4 or later
  3. GNU/Linux and X11

Compilation & Installation

  1. Additional requirements: installed kdelibs-dev package
  2. From the top source directory, run "./Setup-kde4.sh"

Qt 4/Linux

Requirements

  1. Qt 4.3
  2. GNU/Linux and X11

Compilation & Installation

  1. From the top source directory, run "./Setup-qt4.sh"

Qt 4/Windows

Requirements

  1. Qt 4.3 source files (static library)
  2. Nullsoft Install System (NSIS)
  3. Windows XP or compatible

Compilation & Installation

  1. From the top source directory, run "Setup-qt4.bat"
